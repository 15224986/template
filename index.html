组件的隐藏事件

组件的公共隐藏事件

可以给任意组件的内部标签添加隐藏事件，只需要给标签上配置data-dismiss="*"属性即可。如果配有 data-dismiss-remove="true" 则是删除，不配则是隐藏。

  • 例如： data-dismiss="panel" 就会找到离他最近的父节点为.panel进行隐藏。
  • 例如： data-dismiss="alert-item" 就会找到离他最近的父节点为.alert进行隐藏。
  • 例如： data-dismiss="tag-item" 就会找到离他最近的父节点为.tag-item进行隐藏。

所以只要不是执行隐藏事件，请不要使用data-dismiss="*"属性。

3.事件
事件 描述 实例
ifHide 当调用 hide 实例方法时立即触发该事件。 
                                                            
                                                                $('.tag-item').on('ifHide', function () {
                                                                    // 执行一些动作...
                                                                })
ifHidden 当提示工具对用户隐藏时触发该事件（将等待 CSS 过渡效果完成）。 
                                                            
                                                                $(document).on('ifHidden', '.tag-item', function(event) {
                                                                    event.preventDefault();
                                                                    // event.target  要操作的dom节点
                                                                    // event.self  点击的按钮本身
                                                                });

改变图标字体文件的位置

Bootstrap 假定所有的图标字体文件全部位于 ../fonts/ 目录内，相对于预编译版 CSS 文件的目录。如果你修改了图标字体文件的位置，那么，你需要通过下面列出的任何一种方式来更新 CSS 文件：

  • 在 Less 源码文件中修改 @icon-font-path 和/或 @icon-font-name 变量。
  • 利用 Less 编译器提供的 相对 URL 地址选项。
  • 修改预编译 CSS 文件中的 url() 地址。

根据你自身的情况选择一种方式即可。
