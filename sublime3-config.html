1.sublime3插件

使用字体
Andale Mono 查看介绍
Package Control
插件管理工具，安装、删除、查看插件。（快捷键）查看介绍
All Autocomplete
匹配css类名。
AutoFileName
路径补全。
autoprefixer
css3前缀补全。查看介绍查看介绍
BracketHighlighter
括号以及标签层级显示，不用担心选中的代码属于哪个代码块，一目了然。
ChineseLocalizations
汉化插件。（语言设置：帮助/Language/简体中文，繁体中文，日本语，English）
CSS Extended Completions
关联CSS文件，智能提示css文件中的类名，非常好用。
CSS3：
CSS扩展完成。
CSScomb
CSS属性排序。
DocBlockr
DocBlokr 帮助你创造你的代码注释，通过解析功能，参数，变量，并且自动添加基本项目。
emmet：
前端神器。一个可以极大提高web开发者HTML和CSS工作效率的工具箱组件。查看介绍
HTML-CSS-JS Prettify
THML、CSS、JS代码格式化，压缩过后的代码可以通过该工具复原。
HTML5
支持hmtl5规范的插件包。
jQuery
为jQuery的大部分方法提供了示例代码段，让jQuery的API更加容易使用。
JavaScript Completions
支持javascript原生语法提示，妈妈再也不用担心我输入document.getElementById(id)。
Sass
SCSS语法高亮包。
SublimeTmpl
快速生成文件模板。
SyncedSidebarBg
自动同步侧边栏底色为编辑窗口底色。
Colorsublime
安装完成后，可输入Install Theme，在线更换主题。主题库有上百种，这个很不错。查看介绍
