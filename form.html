Cascader 级联选择器

当一个数据集合有清晰的层级结构时，可通过级联选择器逐级查看并选择。

  • 里面共有2个 input 分别用来存放label和value的值。
  • 可以通过 name 给里面的input配置name。
  • 可以通过 data-options="数组" 来进行配置data。类型：Array
  • 可以通过 data-value="['beijing', 'gugong']" 来进行配置默认值。类型：Array
  • 可以通过 disabled 来进行配置是否可用。
  • 可以通过 changeSelect="true" 点选每级菜单选项值都会发生变化。
  • 可以通过 trigger="hover" 次级菜单展开方式，可选值为 click 或 hover。

回调事件

提供了大量回调事件，都可以用来监听change事件。

  • ifClicked 用户点击了自定义的输入框或与其相关联的label
  • ifChanged 输入框的 checked 或 disabled 状态改变了
  • ifChecked 输入框的状态变为 checked
  • ifUnchecked checked 状态被移除
  • ifDisabled 输入框状态变为 disabled
  • ifEnabled disabled 状态被移除
  • ifCreated 输入框被应用了 mocCheck 样式
  • ifDestroyed mocCheck 样式被移除

使用on()方法绑定事件

ifCreated 事件应该在插件初始化之前绑定。

                                                $('input').on('ifChecked', function(event){
                                                    alert(event.type + ' callback');
                                                });

方法

下面这些方法可以用来通过编程方式改变输入框状态（可以使用任何选择器）：

  • $('input').mocCheck('check'); 将输入框的状态设置为checked
  • $('input').mocCheck('uncheck'); 移除 checked 状态
  • $('input').mocCheck('toggle'); toggle checked state
  • $('input').mocCheck('disable'); 将输入框的状态设置为 disabled
  • $('input').mocCheck('enable'); 移除 disabled 状态
  • $('input').mocCheck('destroy'); 移除 mocCheck 样式

      使用方法及原理

      • 新增的按钮类名是clone-add-btn，通过for(href)获取到所要复制的内容的模板，for(href)里面是模板的id。
      • 通过按钮向上查询，获得[location]节点，这个是要添加到的位置。可以传递参数("before"、"after"、"append"、"prepend")，默认是这"before"
      • 模板必须包含类名是"clone-template"(隐藏模板)，里面含有从新调用插件的，在原类名上跟随-template。例如：class="select-2" 变为class="select-2-template"

      删除本行和复制本块

      使用方法及原理

      • 删除按钮的类名是clone-remove-btn，向上查找（父级），找到最近的带有self-is-clone类名的父级，进行删除。

      复制 and 删除 的回调

      使用方法及原理

      • 新增 and 删除的回调：data-beforeCallback新增之前（删除之前）的回调，新增有两个参数（self，cloneDom）按钮本身 and 赋值出来的结构。删除只有一个参数（self）按钮本身。
      • 新增 and 删除的回调：data-afterCallback新增之后（删除之后）的回调，有一个参数（self）按钮本身。
